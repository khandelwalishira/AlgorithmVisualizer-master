

# Algorithm Visualizer



So far there are 4 segments  

- Prime Numbers
- Sorting Algorithms
- N Queen
- Binary Search Game

  

## Algorithms implemented 


- Bubble sort
- Selection sort
- Insertion sort
- Heap sort
- Merge sort
- Quick sort
- Sieve of Eratosthenes
- N Queen Backtracking
- Binary Search
